Object system implementation for PHP

see `poem.ja.md`.

Installation
------------

### Composer

```
composer require zonuexe/objectsystem
```

Features
--------

* Property type check at run time
  * `trait TypedProperty`
  * `trait TypeAssert`
  * `class TypeDefinision`
