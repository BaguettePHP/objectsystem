Teto Objectsystem
=================

Object system implementation for PHP

see `poem.ja.md`.

Installation
------------

### Composer

```
composer require zonuexe/objectsystem
```

Features
--------

* Dynamic typing for PHP object
  * trait `TypedProperty`
  * trait `TypeAssert`
  * class `TypeDefinision`
